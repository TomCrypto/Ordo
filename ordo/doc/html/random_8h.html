Detailed Description

OS-provided CSPRNG interface.

Exposes the Ordo CSPRNG (Cryptographically Secure PseudoRandom Number Generator) interface, which is basically a cross-platform wrapper to the OS-provided entropy pool.

Linux: Reads from /dev/urandom
Windows: Acquires a CSP token and calls CryptGenRandom.
