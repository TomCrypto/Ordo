Detailed Description

Ordo test driver interface.

This is the interface to the Ordo test driver, which features a simplistic test vector script language, and functions to parse it.

void displayEnvironmentInfo ( )

This function prints out environment information to the standard output. This includes the platform on which the code is currently running, the word size (32/64 bit), the ABI in use (this is inferred) and CPU feature flags such as support for SSE instructions, AES-NI, etc...
