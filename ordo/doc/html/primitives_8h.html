Detailed Description

Cryptographic primitive interface.

Exposes the Ordo primitive interface, which abstracts various cryptographic primitives through high-level interfaces. Note it is always possible to skip this API and directly access the lower-level primitive functions, but it is discouraged.

See also:
primitives.c

void cipherFree ( CIPHER_PRIMITIVE_CONTEXT *  ctx)

This function frees (deallocates) an initialized cipher primitive context.

Parameters:
ctxThe cipher primitive context to be freed. This context needs to at least have been allocated.
Remarks:
Once this function returns, the passed context may no longer be used anywhere and sensitive information will be wiped. Passing zero to this function is invalid and will incur a segmentation fault. Do not call this function if cipherCreate failed.
