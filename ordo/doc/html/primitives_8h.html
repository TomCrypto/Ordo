Detailed Description

Cryptographic primitives.

This declares all the cryptographic primitives in the library, abstracting various categories through high-level interfaces.

void blockCipherFree ( BLOCK_CIPHER_CONTEXT *  ctx)

This function frees (deallocates) an initialized block cipher context.

Parameters:
ctxThe block cipher context to be freed. This context needs to at least have been allocated.
Remarks:
Once this function returns, the passed context may no longer be used anywhere and sensitive information will be wiped. Passing zero to this function is invalid and will incur a segmentation fault. Do not call this function if blockCipherCreate() failed, as the latter already works hard to ensure no memory is leaked if an error occurs.
