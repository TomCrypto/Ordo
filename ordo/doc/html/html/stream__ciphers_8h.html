Detailed Description

Abstraction Layer.

This abstraction layer declares all the stream ciphers and also makes them available to higher level modules. This does not actually do encryption at all, but only abstracts stream cipher permutations, the encryption modules are in the enc folder: enc_stream.h.

[in]lenThe length, in bytes, of the buffer.

This function is stateful and will update the passed state (by generating keystream material), unlike block ciphers, which are deterministic permutations.
