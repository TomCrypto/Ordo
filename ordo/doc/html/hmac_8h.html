Detailed Description

HMAC interface.

Interface to compute HMAC's (Hash-based Message Authentication Codes), which combine a hash function with a cryptographic key securely to provide both authentication and integrity, as per RFC 2104.

void hmacFree ( HMAC_CONTEXT *  ctx)

This function frees (deallocates) an initialized HMAC context.

Parameters:
ctxThe HMAC context to be freed. This context needs to at least have been allocated.
Remarks:
Once this function returns, the passed context may no longer be used anywhere and sensitive information will be wiped. Passing zero to this function is invalid and will incur a segmentation fault. Do not call this function if hmacCreate() failed, as the latter already works hard to ensure no memory is leaked if an error occurs.
