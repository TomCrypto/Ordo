Detailed Description

Null Cipher block cipher.

This cipher is only used to debug the library and does absolutely nothing, in other words, it is the identity permutation. It accepts any key size but does not even attempt to read the key, and has no parameters. Its block size is 128 bits and is arbitrarily chosen.
