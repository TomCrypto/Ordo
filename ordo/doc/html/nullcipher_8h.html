Detailed Description

NullCipher cipher interface.

Contains the NullCipher cipher primitive interface. This cipher is compatible with all encryption modes of operation and is only used to debug the library. It accepts no parameters, and has an arbitrarily-chosen 128-bit block size. It accepts any key size but does not even attempt to read the key.
