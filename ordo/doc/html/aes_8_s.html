Detailed Description

AES assembly code [ABI: Windows x64, Linux x64].

This file contains an implementation of AES (Advanced Encryption Standard) for AES-NI-capable processors. This technically works for 32-bit processors too, but the x86 version doesn't seem to recognize the forward function declarations and complains about undefined references...
