Detailed Description

Threefish-256 block cipher interface.

Contains the Threefish-256 cipher primitive interface.

Threefish-256 is a block cipher, which has a 256-bit block size and a 256-bit key size. It also has an optional 128-bit tweak. The tweak can be set through the cipher parameters.
