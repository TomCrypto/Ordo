Detailed Description

Hash function interface.

Interface to compute cryptographic digests, using hash functions. This is a very thin wrapper around the low-level functions, since hash functions are quite stateless (they do not require many parameters).

void hashFunctionFree ( HASH_FUNCTION_CONTEXT *  ctx)

This function frees (deallocates) an initialized hash function context.

Parameters:
ctxThe hash function context to be freed. This context needs to at least have been allocated.
Remarks:
Once this function returns, the passed context may no longer be used anywhere and sensitive information will be wiped. Passing zero to this function is invalid and will incur a segmentation fault. Do not call this function if hashFunctionCreate() failed, as the latter already works hard to ensure no memory is leaked if an error occurs.
